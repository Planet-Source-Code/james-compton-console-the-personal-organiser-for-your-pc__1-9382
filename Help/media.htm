Media

This section allows you to play various types of media such as video and sound files. Mp3's, waves. mpegs, midi, avi and many more file types can be played. It also plays Winamp playlists, however it will not display the current track name or number.

List loads an AVP playlist (Audio/Video Playlist) which are the file types that Console supports for it's own playlists. This works in the same way as load, except only valid playlists are allowed to be loaded. AVP playlists can consist of Audio or Video as the name suggests and displays track info at the bottom of the screen.

'RND' is for AVP playlists ONLY! It simply plays a random track in the playlist when the track has finished, or the user selects next or previous.

To play a video in full screen hit ALT+ENTER.

Other options can be got to by right-clicking on the video window (the one with the Windows logo on it).
