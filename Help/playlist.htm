Playlist

Here you can create your own playlists for use in the media player. It works with AVP playlists, which are unique to this program. The great thing is that you can have videos in your playlists too! The menu options are on the right of the playlist window. 'Add' adds all selected files from the file window to the playlist window. You can also do this by double-clicking on the file in the file window. To select more than one file at once, use SHIFT or CTL. while clicking. This only works with the file window. 'Clear' wipes all data that is in the playlist without asking you to save before hand - so make sure you do! 'Move Up' and 'Move Down' simply move the selected track up or down the playlist. Useful for arranging it just how you want it. 

'Transfer' copies all the playlist data from the list to the media player and starts playing at track 1 unless 'RND' was selected in the media player, in which case it will play a random track.

Lastly, if you right click on a file in the playlist, a small popup menu appears. The options are 'Play' or show 'Info'. 'Play' transfers the playlist to the media player, then begins playing on the selected track even if 'RND' was selected. 'Info' shows a little message box with file details such as track number, total and full filename.
